# Schedule
*component documentation*
## Overview
A schedule display module, will display the classes got from database
## API
### Props
- `view`
	- **Type:** bool
	- **Description:** display daily / weekly classes
	- **Default:** true
- `classes_today`
	- **Type:** class[]
	- **Description:** today's classes
	- **Default:** []
- `classes`
	- **Type:** class[]
	- **Description:** this week's classes
	- **Default:** []
